# Lets Get It Dude

Simple Python workspace initialized with a minimal package structure.

## Structure

- `Lets_Get_It_Dude.py` - starter script
- `my_portfolio/` - package directory

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Notes

Add your portfolio code under `my_portfolio/` and update this README as the project grows.
