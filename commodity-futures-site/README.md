# Commodex Futures

A starter website for commodity futures trading insights and simulated trade planning.

## Structure

- `index.html` — landing page and market overview
- `styles.css` — visual styling for the site
- `app.js` — simple simulated market price updates and trade calculator

## Run locally

Open `commodity-futures-site/index.html` in your browser.

## Notes

This site is a demo and does not execute real trades. For production, add a backend API, broker integration, and secure user authentication.
